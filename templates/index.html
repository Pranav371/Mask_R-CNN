About Mask R-CNN

Mask R-CNN is a deep neural network that performs instance segmentation by recognizing objects in an image and generating a high-quality segmentation mask for each instance.

This demo uses a pre-trained Mask R-CNN model implemented in PyTorch, trained on the COCO dataset which can detect 80 different object categories.

For videos, processing may take some time depending on the video length and complexity.
