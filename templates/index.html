About Mask R-CNN
Advanced Computer Vision Technology

Mask R-CNN is a state-of-the-art deep neural network that performs instance segmentation by detecting objects in an image and generating a high-quality segmentation mask for each instance.

This application uses a pre-trained Mask R-CNN model implemented in PyTorch, trained on the COCO dataset which can detect 80 different object categories with remarkable accuracy.

For videos, processing may take some time depending on the video length and complexity. For optimal performance, a GPU is recommended.
